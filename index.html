About

Hello, I'm Catherine.

I am a designer turned software engineer who loves thinking about the connection between technology and the human mind.

Prior to becoming a software engineer, I was a fashion designer - very much immersed in the world of creativity; however, I had always been passionate about technology - having taught myself how to create "choose your own adventure" type of games on MS-DOS at 8 years old, and learning HTML and CSS at the age of 12.

I enjoy problem-solving, building practical systems and products, and even figuring out the annoying little bugs. The constant challenge and the never-ending list of new technologies to learn is exciting for me. In my free time, I love baking, dabbling in ceramics, spending time in nature, and playing with my 10 year old Yorkie named Chloe.

Profferly
Live | Github
Profferly is a volunteer app born during the COVID-19 pandemic. As a platform for the community formed around lending a hand, it provides volunteers with opportunities to connect with one another.
React / Redux / Sass / MongoDB / Express.js / Node.js / JavaScript / HTML5 / CSS3
Depth Perception
Live | Github
Depth Perception is a meditation visualization app where the user input dissipates to a timed animation. The duration of the animation will depend on the length of the chosen meditation sound. It is the idea of letting go of your thoughts to reach deeper meditation.
JavaScript / HTML5 / CSS3 / Sass / Anime.js
Wordplay
In progress | Github
Wordplay is a single-page web application, built on React-Redux and RESTful JSON Rails API. It features an ability for users to highlight lines of spoken word and poetry, and save annotations to the selection. It is a clone of the Genius app.
React / Redux / Ruby on Rails / JavaScript / HTML5 / CSS3 / Sass / AWS S3
